### Key Observations

1. **Large train-val gap on soundscapes.** Training soundscape ROC-AUC (0.992) vs validation (0.665) shows a ~0.33 gap. This is the dominant signal to investigate — the probe may be overfitting to training soundscape structure or the validation set may represent a different acoustic distribution.

2. **Prior fusion provides minimal lift.** val_prior_fusion (0.662) is slightly below raw val_soundscape (0.665). The Bayesian prior is not yet calibrated well enough to help. The OOF probe ROC-AUC (0.520) is near random for 75 classes, confirming the probe still needs work.

3. **padded_cmap is very low (0.062).** This metric is sensitive to temporal localization. The embedding-probe approach may be producing good class rankings but poor temporal resolution, or the padding scheme is penalizing predictions heavily.

4. **Only 52 of 75 classes are fitted.** 23 classes lack sufficient data for the probe, which will produce default/zero predictions for those classes and hurt macro-averaged metrics.

### Root Cause

The pipeline is end-to-end functional but the embedding probe generalizes poorly from training to validation soundscapes. The near-random OOF probe AUC (0.520) confirms the linear probe on Perch embeddings is underfitting or the features lack discriminative power at current dimensionality. The train-val gap (0.33) is the single most important number to reduce.

### Immediate Next Steps

1. **Research stage:** Investigate probe architecture improvements (deeper head, class-specific thresholds), embedding normalization, and calibration methods (temperature scaling, Platt).
2. **Address class coverage:** Strategies for the 23 unfitted classes — zero-shot transfer from Perch label space, upsampling, or pseudo-labeling.
3. **Submission bundle parity:** Package the current pipeline into a CPU-only, internet-off notebook to establish a leaderboard baseline.
4. **Calibration:** The prior fusion needs recalibration before it can improve over the raw probe output.
